| --- | --- | --- | --- |
| 沟通的要素 | 观察—感受—需要—请求<br/>做>听>说<br/>阐释自己的观察是沟通中最先行的 | | |
| 异化沟通的方式 | 道德评判<br/>进行比较<br/>回避责任<br/>强人所难 | | |
| 沟通的要点 | 区分观察与评论<br/>区分想法与感受<br/>表达自身感受<br/>理解对方感受与需求<br/>请求而不是命令<br/>给予反馈<br/>学会倾听 | | |
| 如何表达愤怒 | 深呼吸<br/>想一想自己为什么愤怒<br/>体会需要表达感受<br/>惩罚会带来坏的结果 | | |
| 如何表达感激 | 对方做了什么—哪些需要得到了满足—自己的心情 | | |
| 非暴力沟通的四个步骤：观察结果而不评判-感受自身情绪而不隐瞒-表达需要而不是指责抱怨-提出具体要求而不是模糊意见 | | | |
